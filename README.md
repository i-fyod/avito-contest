# Веб-приложение для модерации объявлений Avito

Это упрощенная версия внутренней системы, предназначенная для модерации объявлений на платформе Avito. Приложение позволяет модераторам проверять объявления, принимать решения по их публикации и отслеживать свою эффективность.

## Технологический стек

### Бэкенд

- **Node.js**: Среда выполнения JavaScript.
- **Express.js**: Легковесный и гибкий фреймворк для создания веб-приложений и API.

### Фронтенд

- **React**: Библиотека для создания пользовательских интерфейсов.
- **TypeScript**: Типизированный JavaScript.
- **React Router DOM**: Для маршрутизации в приложении.
- **Mantine (Core, Hooks, Charts, Form)**: Набор React-компонентов и хуков.
- **Recharts**: Библиотека для построения графиков и диаграмм.
- **Lucide React**: Библиотека иконок.
- **Axios**: HTTP-клиент для взаимодействия с API.
- **Vite**: Быстрый сборщик проектов для фронтенда.

## Начало работы

Следуйте этим инструкциям, чтобы развернуть и запустить проект на вашей локальной машине.

### Предварительные требования

Для запуска проекта вам потребуется установленный Node.js.

- **Node.js**: `v20.x` или выше.

### Установка

1.  Клонируйте репозиторий:

    ```bash
    git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
    cd avito-contest
    ```

2.  Установите зависимости для бэкенда:

    ```bash
    cd backend
    npm install
    cd ..
    ```

3.  Установите зависимости для фронтенда (используется `yarn` из-за `packageManager` в `package.json`, но `npm` также сработает):
    ```bash
    cd frontend
    yarn install # или npm install
    cd ..
    ```

### Запуск приложения

1.  Запустите бэкенд-сервер:

    ```bash
    cd backend
    npm start # Для продакшн режима или npm run dev для разработки с автоматической перезагрузкой
    cd ..
    ```

    Сервер будет доступен по адресу `http://localhost:3000`.

2.  Запустите фронтенд-приложение:
    ```bash
    cd frontend
    yarn dev # или npm run dev
    cd ..
    ```
    Фронтенд будет доступен по адресу `http://localhost:5173` (или другому порту, указанному Vite).
